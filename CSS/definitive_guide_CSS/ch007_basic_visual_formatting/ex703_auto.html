auto value는 width와 margin-left, margin-right에만 줄 수 있다.

.parent

.content {width: auto; margin: auto}


.parent2

.content2 {width: 400px; margin: auto}


auto로 지정된 property를 포함한 7 properties(2 margins, 2 borders, 2 paddings, 1 content width)의 합은 parent element의 width와 같아지도록 auto의 값이 결정된다.
아래의 예와 같이 parent element의 length가 500px일 때, auto로 지정된 margin-left의 value는 200px(500 - width - right margin = left margin)가 된다.

.parent3 {width: 500px}

.content3 {
margin-left: auto;
margin-right: 100px;
width: 200px
}

3가지 property 모두가 auto가 아닌 특정한 값으로 지정될 경우, (left-to-right text direction인 경우) right-margin에 준 값은 의미가 없다

.parent3 {width: 500px}

.content4 {
margin-left: 100px;
margin-right: 100px;
width: 100px
}

반대로 (right-to-left text direction인 경우)는 left-margin에 준 값은 의미가 없다

.parent3 {
width: 500px;
direction: rtl
}

.content4 {
margin-left: 100px;
margin-right: 100px;
width: 100px
}

양쪽 마진이 명시적으로 설정되고, width가 auto로 된다면 width 값은 (부모 요소의 콘텐츠 너비로 요구되는) 임의의 값으로 설정된다.

.parent3 {width: 500px}

.content5 {
margin-left: 100px;
margin-right: 100px;
width: auto
}

양쪽 마진만 지정하고 width는 지정하지 않으면 위의 경우와 같다

.parent3 {width: 500px}

.content6 {
margin-left: 100px;
margin-right: 100px;
}

margin과 width를 auto로 설정하면 auto로 지정된 margin은 0으로 결정된다. width는 자신이 포함되는 block을 채우기 위해 필요한 값으로 설정된다.

.parent3 {width: 500px}

.content7 {
margin-left: auto;
margin-right: 100px;
width: auto }

세 가지의 property 모두 auto로 설정하면 양쪽 마진이 0으로 설정되고 width는 가능한 만큼 넓어진다. 이 결과는 margin이나 width에 대해 아무런 값도 정의되지 않을 때와 같아진다.

.parent3 {width: 500px}

.content8 {
margin-left: auto;
margin-right: auto;
width: auto }

부모 요소의 패딩, 보더, 마진이 자식요소에 영향을 미칠 수 있다.

.parent4 {width: 500px; padding:30px;}

.content9 {
margin: 20px;
padding: 20px;
}
