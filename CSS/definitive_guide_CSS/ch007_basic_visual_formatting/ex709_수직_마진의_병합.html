수직으로 인접한 마진 간의 병합

  • li { margin-top: 10px; margin-bottom: 15px; }
  • li { margin-top: 10px; margin-bottom: 15px; }
  • li { margin-top: 10px; margin-bottom: 15px; }
  • li { margin-top: 10px; margin-bottom: 15px; }

margin이 중첩될때 하나로 병합된다.

  • ul.ex2 { margin-bottom: 15px } ul.ex2 li { margin-bottom: 20px }

h4 { margin-top: 28px }

Containing block에 border나 padding을 추가하는 경우, li는 ul에 포함되므로 ul의 margin과 h1의 margin만 병합된다.

  • ul.ex3 { margin-bottom: 15px; border: 1px solid; } ul.ex2 li { margin-bottom: 20px }

h4 { margin-top: 28px }
