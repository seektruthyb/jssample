이 문단은 { text-align: left; }의 스타일이 적용되어 있는데, 이는 요소의 라인 박스를 문단의 왼쪽 끝으로 정렬 시킨다.

이 문단은 { text-align: right; }의 스타일이 적용되어 있는데, 이는 요소의 라인 박스를 문단의 오른쪽 끝으로 정렬 시킨다.

이 문단은 { text-align: center; }의 스타일이 적용되어 있는데, 이는 요소의 라인 박스를 문단의 중앙으로 정렬 시킨다.

이 문단은 { text-align: justify; }의 스타일이 적용되어 있는데, 이는 요소를 문단의 양쪽 끝으로 정렬 시킨다. 맨 마지막의 행은 예외적인데, 오른쪽 끝이 문단의 오른쪽으로 정렬되지 않는다(오른쪽에서 왼쪽으로 읽고 쓰는 언어에서는 마지막 행이 왼쪽끝으로 정렬되지 않을 것이다).
